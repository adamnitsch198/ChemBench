Capabilities
01
Multi-peak identification
Enter multiple chemical shifts simultaneously. Cross-references all entries to identify compounds matching the full peak pattern.
02
Confidence scoring
Each candidate receives a percentage confidence score derived from ppm proximity, multiplicity agreement, and peak set coverage.
03
Spectrum visualisation
Interactive Lorentzian spectrum rendering with correct multiplicity patterns - singlets through septets - with hover identification.
04
Stabiliser database
BHT, amylene, ethanol, propylene oxide, MEHQ, triethylamine and more - each tagged to their parent solvent.
05
Drying agent compatibility
Functional group-aware recommendations with ranked output, hazard assessment, dosage and procedural guidance.
06
Peer-reviewed data
All reference data drawn from 10 primary peer-reviewed sources including Fulmer, Whiteker, Gottlieb, Burfield, and Pangborn.
